Comparativa de editores de texto

Tabla comparativa de las características de los editores de texto
Editor de texto Características
Aptana En general es un entorno de desarrollo basado en Eclipse fiable y versátil, aún así se ha visto sobrepasado a lo largo de los años por otros editores.

Respecto a su interfaz, es muy configurable, y su integración con Git es un bonus a tener en cuenta. La consola integrada y la vista general están bien logradas,
proporciona la mayoría de cosas necesarias para empezar a trabajar en un entorno de desarrollo local.

Algunos de los menús, heredados de Eclipse, pueden resultar incómodos al uso, no ha sido actualizado desde hace unos años y no ofrece soporte para versiones de PHP superiores a la 5.4.
Brackets Editor creado por Adobe, muy enfocado a HTML y CSS, además de JavaScript. Cuenta con algunas facilidades interesantes para aquellos que son principiantes, véase:
  • Live preview: mediante una combinación de teclas es capaz de mostrarte la definición de una etiqueta en CSS o de una función JavaScript directamente en línea,
    sin cambiar de editor, permitiéndote además editarla.
  • Vista Previa de los campos en tiempo real: herramienta específica para visualizar cambios a tiempo real en el navegador sin necesidad de refrescar ni guardar el documento.
  • Soporte de serie para preprocesadores CSS.
Atom Es el editor de código de la plataforma de desarrollo Github, es compatible con todos los sistemas operativos, y ofrece muchas posibilidades de extensión.

Está estrechamente ligado con Github y permite establecer un repositorio GitHub para el proyecto sin tener que visitar la plataforma, entre otras posibilidades.
Puedes también acceder al sistema de control de versiones Git para tener controlados todos los cambios efectuados en el código.

Un pequeño inconveniente puede ser que la interfaz solo se halla en inglés.
Notepad++ Su mayor ventaja es, además de ser gratuito como todos los editores de esta tabla, que está escrito en lenguaje C++.
Ofrece las características básicas de un editor de código HTML, cuenta con un editor de macros incorporado por defecto y una función autocompletar,
además permite visualizar previamente los documentos HTML en los navegadores instalados en el PC.

Su mayor inconveniente es que puede llegar a tener problemas con archivos muy grandes.
Sublime Text Siendo un editor de texto bastante ligero, proporciona al usuario algunas características provenientes de los IDE, la opción de modificar cada aspecto del editor en sí,
dejando a los usuarios escribir código con rapidez y eficiencia. Tiene un minimapa que provee una vista de arriba abajo del archivo junto con atajos para la mayoría de acciones,
soporta un gran número de lenguajes y funcionalidades generales de edición de texto.

Podría decirse que es indicado para principiantes, al ser rápido y fácil de instalar.
Visual Studio Code Actualmente el más utilizado, gratuito, de código abierto, multiplataforma, rápido, potente y con una gran capacidad de ampliación mediante extensiones,
integra soporte para las herramientas más habituales como Git o Docker.

También soporta infinidad de lenguajes, ya sea de manera nativa o mediante extensiones, y gracias a la herramienta Live Share, introducida a finales del 2017,
los desarrolladores son capaces de utilizar Visual Studio o Visual Studio Code para colaborar en tiempo real.



Finalmente, observando las opciones propuestas y analizando sus características, me quedo con el Visual Studio Code,
ya que lo veo el más apropiado a largo plazo, contando con un sinfín de herramientas, soporte y actualizaciones recientes,
además de su masiva comunidad de usuarios en la que poder consultar cualquier duda.

Sin embargo si tuviese que elegir alguno específicamente dedicado para mi familiarizacion y aprendizaje con los editores
de texto y entornos de desarrollo para aplicaciones web, lo más seguro es que me quedaría con Brackets o Sublime.
